Welcome to my blog! This is a sample post that doubles as a **template** — to write a new post, copy this file inside `src/content/blog/`, rename it (the file name becomes the URL), and edit the frontmatter at the top.

## What goes in the frontmatter

- `title` — the post title (required)
- `date` — publish date in `YYYY-MM-DD` format (required)
- `description` — a short summary shown on the blog index (optional)
- `tags` — a list of tags (optional)

## Markdown works out of the box
